## Tech Stack

- **React 19** + **TypeScript**
- **Tailwind CSS 4** + **class-variance-authority** (variants)
- **Radix UI** (headless primitives)
- **Storybook 9** (documentation + visual testing)
- **Vitest** (unit testing)
- **tsup** (ESM/CJS build)
- **Turborepo + pnpm** (monorepo)
- **GitHub Packages** (private npm registry)

---

## Installation

```bash
pnpm add @redotlabs/ui @redotlabs/tokens @redotlabs/themes
```

Requires GitHub Packages registry in `.npmrc`:

```
@redotlabs:registry=https://npm.pkg.github.com
```
