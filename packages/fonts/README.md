# Font packages for Redot design system

## Included Fonts

- **Pretendard**: Supports weights 100-900

## Usage

```tsx
// main.tsx, index.tsx ...

export default function App() {
  return (
    <ThemeProvider color="blue" font="pretendard">
      <></>
    </ThemeProvider>
  );
}
```

## Font Source

All fonts are served from Redot CDN (`https://cdn.redot.me/fonts/`).
